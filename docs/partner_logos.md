# Partner Logos

Styling for .partner-logos, .text-partner-logos

## hover
On default and mobile, figcaption is hidden. 
On hover figcaption is visible on desktop. 

```
figure:hover figcaption {
    visibility: visible;
    opacity: 1;
}
```
